--------------------------Student Management System---------------------------------

1. Unzip the project folder.
2. Open the XAMPP control panel and start the MYSQL and APACHE servers.
3. Project folder open in visual code.
4. Open the terminal and run the code 'pip install flask'.
5. Create the SQL database including users and students tables.
6. Then run the code 'python app.py' in terminal.
7. Open browser and search the 'localhost:8080/register'.
